# MoodApp - 您的情绪伙伴与安全卫士

🌟 **记录心情，守护安全，简单而强大的个人关怀工具。**

MoodApp 是一款专为 iOS 用户设计的原生应用，旨在通过极简的操作，帮助您追踪情绪变化，并为您的人身安全提供一份额外的保障。无论是记录生活中的点滴快乐，还是在忙碌中确保一份“我安好”的承诺，MoodApp 都是您最贴心的陪伴。

---

## ✨ 核心功能

### 🌈 心情追踪：捕捉您的每一刻
- **表情签到**：只需一秒，点击对应表情（😔😐🙂😄）即可完成记录。
- **趋势分析**：内置精美图表，自动生成本周心情分布，帮您发现情绪规律，更好地了解自己。

### 🛡️ 安全签到：24小时的无声守护
- **“我安好”按钮**：每天只需轻轻一点，即可重置 24 小时安全倒计时。
- **智能预警**：若超过 24 小时未签到，系统将发出温馨提醒，确保您的状态始终处于关注之中。

### 📞 紧急联系：关键时刻的桥梁
- **一键添加**：直接从通讯录导入最信任的 3 位联系人。
- **快速测试**：支持发送测试通知，确保在真正需要时，联系通道畅通无阻。

---

## 💎 为什么选择 MoodApp？

- **🔒 隐私至上**：我们深知隐私的重要性。所有数据均存储在您的手机本地（SwiftData），**不上传云端，无需注册账号**，您的隐私只属于您自己。
- **🚀 极简原生**：基于 SwiftUI 开发，完美适配 iOS 系统。启动飞快，操作顺滑，支持**暗黑模式**，无论昼夜都给您最舒适的视觉体验。
- **📊 深度统计**：不仅记录，更懂分析。提供心情走势图与签到完成率，见证您的成长与坚持。
- **📂 数据自主**：支持将您的心情与签到数据导出为 CSV 或 JSON 格式，方便随时备份或分析。

---

## 🚀 快速上手

1. **首次启动**：授权通知权限，以便接收安全提醒。
2. **记录心情**：在首页点击「记录心情」，选择今天的状态。
3. **确认安全**：每天点击一次「我安好」，保持您的安全状态。
4. **添加家人**：进入「紧急联系」，添加 1-3 位紧急联系人并测试通知。

## 📖 使用指南

### 如何设置提醒时间？
在「系统设置」中，您可以根据作息习惯自定义每日心情签到的提醒时间（默认为上午 09:00）。

### 数据不小心删了怎么办？
我们在「系统设置」中提供了数据管理功能，支持导出备份。请注意，如果您选择了「清除记录」或「注销账号」，本地数据将无法找回。

### 为什么我收不到安全提醒？
请确保在手机系统的「设置」-「通知」中允许了 MoodApp 的权限。此外，建议每天打开一次应用，以确保后台任务能够正常运行。

## ❓ 常见问题 (FAQ)

**Q: 这个应用收费吗？**
A: MoodApp 的核心功能完全免费，旨在服务每一位关注心理健康与安全的用户。

**Q: 我的联系人会收到骚扰吗？**
A: 绝对不会。MoodApp 仅在您手动点击「测试通知」时才会发送测试消息。在未来的版本中，只有在极度异常的情况下才会引导触发警报。

**Q: 支持导出数据吗？**
A: 支持。在「数据统计」页面，您可以一键导出 CSV 格式的心情数据。
